## VISUAL STYLE REFERENCE

### Available Colors
**Dark Backgrounds (default):**
- Primary text: `cream` (light beige)
- Secondary text: `cream/70`, `cream/60`
- Accents: `purple`, `green`, `sage`, `taupe`, `gold`

**Light Card Backgrounds:**
- Neutral: `neutral-50`, `neutral-100` with `neutral-700/900` text
- Green: `green-50` with `green-700/900` text
- Purple: `purple-50` with `purple-700/900` text

### Icon Reference (Semantic Names)
Common icons from the registry:
- **Tech:** `code`, `database`, `api`, `terminal`, `cpu`
- **AI/Data:** `ai`, `brain`, `network`, `chart`, `analytics`
- **Business:** `growth`, `target`, `rocket`, `briefcase`, `trending`
- **Science:** `dna`, `flask`, `microscope`, `atom`
- **Security:** `shield`, `lock`, `check`
- **Health:** `heart`, `activity`, `pill`
- **Misc:** `info`, `alert`, `star`, `zap`

### Layout Patterns

**Grid Layouts:**
```
2-column: Good for comparisons (Problem/Solution, Before/After)
3-column: Good for process steps or categories
4-column: Good for statistics or key metrics
```

**Card Styles:**
```
Dark transparent: bg-white/5 border-cream/10
  → Best for maintaining dark background consistency

Light solid: bg-green-50 border-green-300
  → Best for emphasis or contrast sections
```

**Callout Boxes:**
```
Left border emphasis: border-l-4 border-[color]-500
Background: bg-[color]-500/10
Label: text-[color]-300 uppercase tracking-wider
```

### Typography Sizes
- Hero/Title: `text-4xl` to `text-6xl`
- Section headings: `text-3xl` to `text-4xl`
- Card headings: `text-lg` to `text-xl`
- Body text: `text-sm` to `text-base`
- Labels/captions: `text-xs`

### Spacing
- Tight: `gap-4`, `space-y-4`
- Medium: `gap-6`, `space-y-6`
- Loose: `gap-8`, `space-y-8`
